📘 إعداد Meta Ads
احصل على Token ↗
💡 كيف تحصل على Token:
1. اذهب لـ developers.facebook.com/tools/explorer
2. اختر تطبيقك → Generate Token
3. اختر صلاحية ads_read وانسخ التوكن
👻 إعداد Snapchat Ads
مساعدة ↗
⚠ Snapchat Marketing API يحتاج موافقة من Snapchat. استخدم CSV كبديل سريع.
الخيار 1 — رفع CSV
📁 اسحب ملف Snapchat CSV أو اضغط للرفع
من Snapchat Ads Manager → Reports → Export CSV
الخيار 2 — Access Token (لو عندك API Access)
🎵 إعداد TikTok Ads
Marketing API ↗
💡 كيف تحصل على Token:
1. اذهب لـ ads.tiktok.com/marketing_api
2. أنشئ App → احصل على Access Token
3. انسخ الـ Advertiser ID من حسابك
أو رفع CSV من TikTok Ads Manager
📁 TikTok CSV
🔍 Google Ads — قريباً
Google Ads API يحتاج Developer Token + OAuth2 Flow.
في الوقت الحالي يمكنك تصدير CSV من Google Ads Manager واستخدامه في تبويب Campaign Data.
📁 Google Ads CSV
